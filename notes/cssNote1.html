Introduction

What is CSS?

  • Language for stylizing HTML files
  • Color, font, weight, etc.

Elements

Link

  • Links .html file to .css file

CSS Rules

Selector

  • Targets the CSS code to the HTML element for stylizing

Property

  • Changes the stylistic feature

Value

  • How the stylistic feature is changed
  • Color, font size, etc.

Common CSS Properties

Font-size

  • Changes text size

Line-height

  • Changes the height of text lines

Color

  • Sets the text color of the selected element
