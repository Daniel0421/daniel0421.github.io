Operators

Basic Arithmetic

  • Can use the +, -, *, / operators to perform basic math functionality
  • Can use arithmetic to perform mathematical function using values of other variables
  • Can update the current value of the variable
  • Double value gives decimal value
  • Division with Int values will always round down to the nearest whole number
  • Can declare type Double for Int operation that require decimal-precision value

Compound Assignment

  • Better way to modify value: add "=" after arithmetic operator

Numeric Type Conversion
