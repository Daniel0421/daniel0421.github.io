URL

  • Universal language of web or webpage

Elements

HTML

  • First of element of any webpage meaning that everything should be inside HTML element
  • HTML tag should end after the webpage is finished

Head

  • Invisible part of the webpage where information is not visible

Body

  • Main feature of the webpage
  • Visible part of webpage

Paragraph

  • Adds paragraphs or any text

Header
